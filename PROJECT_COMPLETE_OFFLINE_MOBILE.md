## 🎓 How It Works

### Online
```
User Opens App
  ↓
Fetch Fresh Data
  ↓
Cache It Locally  ← For offline use
  ↓
Display All Features
```

### Offline
```
User Opens App (No Internet)
  ↓
Load from Cache
  ↓
"📵 Offline Mode" Badge Shows
  ↓
Limited But Functional
  ↓
Reports Queue Locally
  ↓
Auto-Sync When Online
```
